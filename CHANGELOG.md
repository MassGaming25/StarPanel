# Changelog

## v1.0.0 — Initial Release

### Features
- Ship database with live data from RSI Ship Matrix / Fleetyards
- Commodity market with best buy/sell locations and trade routes
- Fleet manager with ship images, purchase tracking, and insurance
- Captain's Log with 8 entry types and searchable location fields
- Auto-updating ship, commodity, and location data from UEX Corp API
- Dark sci-fi HUD theme

### Data Sources
- Ships: Fleetyards API → RSI Ship Matrix (fallback)
- Commodities & Prices: UEX Corp API
- Locations: UEX Corp terminals endpoint
